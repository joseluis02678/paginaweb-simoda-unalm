## 📋 Descripción

El **Semillero de Investigación SIMODA** (Statistics, Data Science e IA aplicada al sector agropecuario) es un círculo de investigación de la UNALM dedicado al desarrollo de proyectos innovadores en:

- 📊 Estadística aplicada
- 🤖 Inteligencia Artificial
- 🌱 Agricultura de precisión
- 📈 Ciencia de Datos
- 💻 Desarrollo de software

Este repositorio contiene el código fuente de nuestra página web institucional, diseñada para mostrar nuestros proyectos, servicios, equipo de trabajo y facilitar el contacto con la comunidad universitaria y el sector agropecuario.

---

## 🚀 Características

- ✨ **Diseño moderno y responsivo** - Adaptado perfectamente a dispositivos móviles, tablets y escritorio.
- 🎨 **Identidad UNALM** - Uso oficial de colores institucionales (verde y dorado) y logos (UNALM, SIMODA, FEYP).
- ⚡ **Alto rendimiento** - Carga ultrarrápida gracias a Vite y optimización de assets.
- 📑 **Secciones completas** - Inicio, Nosotros, Proyectos, Servicios, Integrantes y Contacto.
- 🔄 **Despliegue continuo (CI/CD)** - Actualizaciones automáticas en Vercel al hacer push a GitHub.

---

## 🛠️ Tecnologías Utilizadas

### Frontend

- **React 19** - Biblioteca para la construcción de la interfaz de usuario.
- **Vite 8** - Herramienta de construcción (build tool) ultrarrápida.
- **HTML5 & CSS3** - Estructura semántica, estilos modernos, Flexbox y CSS Grid.
- **JavaScript (ES6+)** - Lógica e interactividad del sitio.

### Herramientas y Despliegue

- **Git & GitHub** - Control de versiones y alojamiento del código.
- **Vercel** - Plataforma de despliegue continuo (Hosting).
- **npm** - Gestor de paquetes de Node.js.
- **Oxlint** - Linter rápido para mantener la calidad del código React.

## 📥 Instalación y Configuración Local

Si deseas ejecutar este proyecto en tu computadora para desarrollo o contribución, sigue estos pasos:

### Prerrequisitos

- [Node.js](https://nodejs.org/) (v20.19.0 o superior)
- [Git](https://git-scm.com/)

### Pasos de instalación

1. **Clonar el repositorio**

   ```bash
   git clone https://github.com/joseluis02678/paginaweb-simoda-unalm.git
   cd paginaweb-simoda-unalm
   ```

2. **Instalar las dependencias**

   ```bash
   npm install
   ```

3. **Ejecutar en modo desarrollo**

   ```bash
   npm run dev
   ```

4. **Abrir en el navegador**

   Visita la URL que te proporciona la terminal, generalmente:

   ```
   http://localhost:5173/
   ```

## 🚀 Comandos Disponibles

| Comando           | Descripción                                                        |
| ----------------- | ------------------------------------------------------------------ |
| `npm run dev`     | Inicia el servidor de desarrollo con Hot Module Replacement (HMR). |
| `npm run build`   | Compila el proyecto para producción en la carpeta `dist`.          |
| `npm run preview` | Sirve la versión de producción localmente para pruebas.            |
| `npm run lint`    | Ejecuta Oxlint para encontrar y corregir problemas de código.      |
